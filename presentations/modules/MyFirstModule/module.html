Project Aim

  • Definition of a multi-layer pathway modelling framework
    • A generic and extendable format for integrating multiple pathway types
    • Easing data integration
    • Increasing reproducibility in the field of network biology
  • System Medicine analyses
    • Increasing interoperability and reproducibility
  • Performance evaluation of current System Medicine methods in cancer data sets

Project Aim

  • N layer
    • Targets one category of the pathways’ components
    • Nodes arranged within layers
  • Inter-/intra-layer edges
    • Defines relation between pathways component
    • e.g: Drug “targets” protein
  • Pathway = set of related nodes
